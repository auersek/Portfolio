# Portfolio

This portfolio is a collection of my programming projects and their descriptions. 

Gomoku.py (done in Python): 
This project is an AI engine for the game Gomoku. Gomoku is played on an 8×8 board. The code prints the Gomoku board and the user takes turns playing against the AI. 

seamcarving.c (done in C) :
Seam-carving is a content-aware image resizing technique where the image is reduced in size by one pixel of width (or height) at a time. A vertical seam in an image is a path of pixels connected from the top to the bottom with one pixel in each row; a horizontal seam is a path of pixels connected from the left to the right with one pixel in each column. 
The files seamcarving.h and c_img.h are also part of this project.

ScoreMax.cpp (done in C++):
ScoreMax is a game where each digit 0-9 is correlated with a score positive or negative. The player is given a sequence of numbers and there are allowed sequences that the player is allowed to make from these numbers. Using all or some of the given numbers the player is meant to create a sequence to achieve the highest score and extra points are awarded for things like using ascending numbers in a row. The code calculates all possible scores and the highest one is selected. There are 4 associated text files with this code: values.txt, allowed.txt, rounds.txt, and output.txt, each of which is read in at the start. The allowed.txt contains the allowed sequences, the values.txt file the associated score, the rounds.txt the sequence given to the player each round. The output.txt file is used to store the maximum score and the corresponding sequence for each round.
